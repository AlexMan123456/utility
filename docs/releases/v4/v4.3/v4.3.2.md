# v4.3.2 (Patch Release)

**Status**: Released

This is a new patch release of the `@alextheman/utility` package. It fixes issues with the package in a way that should require no refactoring. Please read below the description of changes.

## Description of Changes

- Adds JSDoc comments to the options interfaces.
    - This was needed to ensure that the options types show up in the generated documentation. Without it, they got skipped, despite the properties of the interfaces showing.
